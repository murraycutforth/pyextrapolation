PDE-based Constant Extrapolation
================================

This Python package is designed to solve the problem of extrapolation of a quantity outwards from the boundary of a region, in an arbitrary number of dimensions. This problem is commonly solved as part of a ghost fluid method for multi-material fluid dynamics.
